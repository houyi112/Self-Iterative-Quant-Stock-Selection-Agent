# 风险因子库

> **用法：** 主Agent做报告一板块预判前，对每个候选板块逐条检测，累加风险分。分值越高 → 预测越保守。具体降多少概率由主Agent结合当日实际情况判断，不做机械扣减。
>
> **分数解读：** 0-2 低风险 | 3-5 中风险（需关注）| 6-8 高风险（需警告）| ≥9 极高风险（建议回避）
>
> **维护：** 复盘Agent每日收盘后根据预测偏差调整因子分值。连续3次无效的因子降权或标记废弃，新发现的风险模式新增因子。
>
> **进化：** 遇矛盾时，调出冲突来源的复盘记录，合并为更精确的因子，废弃旧的。

## 因子列表

### F001 · 连涨超买
板块连续上涨天数过长，技术面超买，回调概率上升。

| 条件 | 加分 |
|------|:--:|
| 板块连续上涨 3-4 日 | **+1** |
| 板块连续上涨 5-6 日 | **+2** |
| 板块连续上涨 ≥7 日 | **+3** |
| MACD DIF 值超过 +100 | **+1** |
| MACD DIF 值超过 +200 | **+2** |

> 单个因子封顶 +5。来源：2026-07-17 医药连涨7日+DIF=+165，次日开盘-2.66%

---

### F002 · 高标情绪断档
市场最高连板高度不足或梯队断层，说明短线情绪脆弱。

| 条件 | 加分 |
|------|:--:|
| 全市场最高连板仅 3-4 板 | **+1** |
| 最高连板 ≥5 板，且独苗（无≥2板的后备梯队） | **+2** |
| 最高标来自目标板块 | **+1**（叠加） |

> 来源：2026-07-17 哈药5连板独苗，医药板块缺乏后备高标

---

### F003 · 逆市补跌
大盘下跌时逆市上涨的板块，次日有补跌风险。

| 条件 | 加分 |
|------|:--:|
| 大盘收跌，该板块逆市上涨 | **+1** |
| 连续 2 个交易日大盘跌而该板块涨 | **+2** |
| 板块逆市涨幅 >2% 但无明确催化剂支撑 | **+1**（叠加） |

> 来源：2026-07-16 大盘-1.85%医药逆市涨，次日补跌-2.66%

---

### F004 · 量价背离
价格上涨但量能跟不上，或放量滞涨，是趋势减弱的早期信号。

| 条件 | 加分 |
|------|:--:|
| 板块连涨但近 2 日成交量递减（涨幅未缩） | **+1** |
| 单日成交量 > 20日均量 2倍，但涨幅 < 1%（放量滞涨） | **+2** |
| 缩量加速上涨（量比 < 0.7 但涨幅 > 昨日涨幅） | **+2** |

---

### F005 · 板块轮动加速
领涨板块频繁切换，说明资金无方向，追高风险加大。

| 条件 | 加分 |
|------|:--:|
| 当日领涨 TOP3 与昨日 TOP3 完全不同 | **+1** |
| 连续 3 日领涨板块无一重复 | **+2** |
| 当日领涨板块盘中冲高回落 >50% 涨幅 | **+1**（叠加） |

---

### F006 · 资金流向背离
价格方向与主力/北向资金方向相反，内部分歧加大。

| 条件 | 加分 |
|------|:--:|
| 板块涨但北向资金对该板块净流出 | **+1** |
| 板块涨但融资余额环比下降 | **+1** |
| 两者同时发生 | **+2**（取代上面两条，不叠加） |

---

### F007 · 技术顶背离
价格创阶段新高但动能指标未同步，趋势衰竭信号。

| 条件 | 加分 |
|------|:--:|
| 价格创 20 日新高但 MACD DIF 未同步新高 | **+1** |
| 价格创 20 日新高但 OBV 或成交量未同步放大 | **+1** |
| 两者同时出现 | **+2**（取代上面两条，不叠加） |

---

### F008 · 利好兑现风险
消息面利好已被价格提前消化，或跳空缺口缺乏实质支撑。

| 条件 | 加分 |
|------|:--:|
| 板块跳空高开 >1% 但无对应力度的实质政策/业绩支撑 | **+2** |
| 盘后出行业利好但盘中板块已拉涨 >2%（利好提前泄露/兑现） | **+1** |
| 个股业绩预告超预期但股价已连续上涨 >5日（price in） | **+1** |

## 使用示例

```
7/17 分析医药板块：
  F001: 连涨7日 +3, DIF>100 +1  →  +4
  F002: 哈药5连板独苗          →  +2
  F003: 7/16逆市已触发          →  +1
  F004: 未触发                  →   0
  F005: 未触发                  →   0
  F006: 未触发                  →   0
  F007: 未触发                  →   0
  F008: 未触发                  →   0
  ─────────────────────────────────────
  总分: 7 → 高风险
  → 主Agent决定：领涨概率从40%降至25%，增加补跌警告
```

## 因子生命周期管理

| 规则 | 说明 |
|------|------|
| **新增** | 从每日复盘偏差中提取可复现的模式 |
| **升级** | 连续 5 次准确预警 → 加分权重 +1 |
| **降权** | 连续 3 次信号发出但未兑现 → 加分权重 -1（最低保留 +0.5） |
| **废弃** | 连续 8 次无效，或市场环境发生结构性变化 |
| **合并** | 两因子高度相关（触发时机重叠 >70%）→ 合并为一条 |
| **封顶** | 总量 ≤30 条，超出时按有效性排名，淘汰末尾 |
